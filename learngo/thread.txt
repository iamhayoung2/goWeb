--> Go Thread함.

go 는 context switching을 줄이고자, os thread를 최소한으로 사용하면서, NM thread (하나의 os 스레드에, 여러개의 go 스레드가 들어갈 수 있음)

go는 컴퓨터가 가진 cpu 갯수만큼 os thread를 만들고, 각 os thread를 잘게 짤라서, 각 go thread로 쓴다.
